
# DB Instance 机型表

高可用型 HA

1) 简介：高可用版提供双主热备架构，避免因宕机或硬件故障造成的数据库不可用，且高可用版使用InnoDB引擎对于主从同步及容灾更佳。
2）内存：支持 1, 2, 4, 6, 8, 12, 16, 24, 32, 48, 64, 96, 128, 192, 256, 320 (单位 GB)
